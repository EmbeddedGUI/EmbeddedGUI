## Theoretical SPI Transfer Performance

Screen: 240x240 px, RGB565 (2 bytes/px), full frame = 115,200 bytes (112.5 KB)

All SPI tests use **dual buffer (DMA double-buffering)**:
CPU renders into buffer A while DMA transfers buffer B, then they swap.
Effective frame time = **max(CPU render time, SPI transfer time)**.
