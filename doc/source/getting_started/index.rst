快速入门
========

本章节面向第一次接触 EmbeddedGUI 的读者，建议按“先跑起来，再理解结构，最后深入架构”的顺序阅读。

- 如果你只想先验证环境，先看 `environment_setup`
- 如果你想尽快跑出第一个界面，先看 `first_app`
- 如果你想理解仓库组织方式，先看 `project_structure`
- 如果你想理解构建系统，先看 `build_system`
