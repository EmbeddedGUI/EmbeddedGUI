# ViewPageCache 示例

## 应用说明

演示 ViewPageCache 控件（缓存翻页容器），支持10页动态加载/释放，左右滑动切页。

## 控件列表

| 控件 | 类型 | 尺寸 (宽x高) | 说明 |
|------|------|--------------|------|
| viewpage_1 | egui_view_viewpage_cache_t | 全屏 | 缓存翻页容器，共10页，动态加载释放 |

## 录制动作

| 序号 | 动作类型 | 方向 | 说明 |
|------|----------|------|------|
| 0 | 滑动 | 向左 | 从屏幕3/4处滑到1/4处，切换到第2页 |
| 1 | 滑动 | 向左 | 从屏幕3/4处滑到1/4处，切换到第3页 |
| 2 | 滑动 | 向右 | 从屏幕1/4处滑到3/4处，切换回第2页 |
| 3 | 滑动 | 向右 | 从屏幕1/4处滑到3/4处，切换回第1页 |
