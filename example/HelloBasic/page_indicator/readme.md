# PageIndicator 示例

## 应用说明

演示 PageIndicator + ViewPage 组合，底部圆点指示器跟随页面切换自动更新。

## 控件列表

| 控件 | 类型 | 尺寸 (宽x高) | 说明 |
|------|------|--------------|------|
| viewpage | egui_view_viewpage_t | 全屏宽 x (屏高-20) | 翻页容器，3页 |
| page_1 | egui_view_label_t | 全屏宽 x (屏高-20) | 第1页，绿色背景，文字"Page 1" |
| page_2 | egui_view_label_t | 全屏宽 x (屏高-20) | 第2页，橙色背景，文字"Page 2" |
| page_3 | egui_view_label_t | 全屏宽 x (屏高-20) | 第3页，蓝色背景，文字"Page 3" |
| indicator | egui_view_page_indicator_t | 全屏宽 x 20 | 页面指示器，3个圆点 |

## 录制动作

| 序号 | 动作类型 | 方向 | 说明 |
|------|----------|------|------|
| 0 | 滑动 | 向左 | 从屏幕3/4处滑到1/4处，切换到第2页 |
| 1 | 滑动 | 向左 | 从屏幕3/4处滑到1/4处，切换到第3页 |
| 2 | 滑动 | 向右 | 从屏幕1/4处滑到3/4处，切换回第2页 |
| 3 | 滑动 | 向右 | 从屏幕1/4处滑到3/4处，切换回第1页 |
