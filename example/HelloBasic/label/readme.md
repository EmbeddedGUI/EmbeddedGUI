# Label 控件演示

## 应用说明

演示 Label 控件基本用法，展示不同尺寸和宽高比的文本标签。包含 4 个标签，分别为宽型和紧凑型两种比例，各有大小两种尺寸。布局采用垂直居中排列。

## 控件列表

| 变量名 | 控件类型 | 尺寸 (宽x高) | 文本内容 | 说明 |
|--------|---------|-------------|---------|------|
| label_1 | Label | 180x40 | "Wide S" | 宽型小标签 |
| label_2 | Label | 220x60 | "Wide L" | 宽型大标签 |
| label_3 | Label | 100x60 | "Tall S" | 紧凑型小标签 |
| label_4 | Label | 140x90 | "Tall L" | 紧凑型大标签 |

## 录制动作

| 序号 | 动作类型 | 间隔 | 说明 |
|------|---------|------|------|
| 1 | WAIT | 1500ms | 等待展示静态标签显示效果 |
