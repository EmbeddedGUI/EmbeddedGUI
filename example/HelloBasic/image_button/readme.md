# ImageButton 示例

## 应用说明

演示 ImageButton 控件（图片按钮），4个不同尺寸的星形图片按钮在2x2网格中。

## 控件列表

| 控件 | 类型 | 尺寸 (宽x高) | 说明 |
|------|------|--------------|------|
| imgbtn_1 | egui_view_image_button_t | 80x40 | 宽+小，星形图片 |
| imgbtn_2 | egui_view_image_button_t | 120x60 | 宽+大，星形图片 |
| imgbtn_3 | egui_view_image_button_t | 45x45 | 方+小，星形图片 |
| imgbtn_4 | egui_view_image_button_t | 80x80 | 方+大，星形图片 |
| grid | egui_view_gridlayout_t | 220x260 | 2列网格布局 |

## 录制动作

| 序号 | 动作类型 | 目标控件 | 说明 |
|------|----------|----------|------|
| 0 | 点击 | imgbtn_1 | 点击图片按钮1 |
| 1 | 点击 | imgbtn_2 | 点击图片按钮2 |
| 2 | 点击 | imgbtn_3 | 点击图片按钮3 |
| 3 | 点击 | imgbtn_4 | 点击图片按钮4 |
