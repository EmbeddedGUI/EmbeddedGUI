# Card 控件演示

## 应用说明

演示 Card 卡片容器控件的基本用法。展示 4 种不同尺寸的卡片，每张卡片内包含标题和内容两个 Label。所有卡片使用圆角半径 8，并设置圆角阴影效果。布局采用垂直水平居中排列。

## 控件列表

| 变量名 | 控件类型 | 尺寸 (宽x高) | 说明 |
|--------|---------|-------------|------|
| card_1 | Card | 180x50 | 宽型小卡片，圆角 8，含阴影 |
| label_title_1 | Label | 160x15 | card_1 标题 "Wide S" |
| label_desc_1 | Label | 160x15 | card_1 内容 "Content" |
| card_2 | Card | 210x70 | 宽型大卡片，圆角 8，含阴影 |
| label_title_2 | Label | 190x20 | card_2 标题 "Wide L" |
| label_desc_2 | Label | 190x20 | card_2 内容 "Content" |
| card_3 | Card | 120x70 | 紧凑型小卡片，圆角 8，含阴影 |
| label_title_3 | Label | 100x20 | card_3 标题 "Tall S" |
| label_desc_3 | Label | 100x20 | card_3 内容 "Content" |
| card_4 | Card | 160x90 | 紧凑型大卡片，圆角 8，含阴影 |
| label_title_4 | Label | 140x25 | card_4 标题 "Tall L" |
| label_desc_4 | Label | 140x25 | card_4 内容 "Content" |

## 录制动作

| 序号 | 动作类型 | 间隔 | 说明 |
|------|---------|------|------|
| 1 | WAIT | 1500ms | 等待展示卡片静态显示效果 |
