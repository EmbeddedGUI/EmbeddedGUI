# Spinner 控件演示

## 应用说明

演示 Spinner 旋转加载控件的基本用法。展示 4 种不同尺寸的旋转指示器，包含宽型和方型两种比例，使用 2 列网格布局（GridLayout）排列。所有 Spinner 在初始化后自动启动旋转动画，无需用户交互。

## 控件列表

| 变量名 | 控件类型 | 尺寸 (宽x高) | 说明 |
|--------|---------|-------------|------|
| grid | GridLayout | 220x220 | 2 列网格容器，居中对齐 |
| spinner_1 | Spinner | 60x30 | 宽型小旋转指示器 |
| spinner_2 | Spinner | 100x50 | 宽型大旋转指示器 |
| spinner_3 | Spinner | 35x35 | 方型小旋转指示器 |
| spinner_4 | Spinner | 70x70 | 方型大旋转指示器 |

## 录制动作

| 序号 | 动作类型 | 间隔 | 说明 |
|------|---------|------|------|
| 1 | WAIT | 1500ms | 等待旋转动画自动播放，捕获旋转效果 |
