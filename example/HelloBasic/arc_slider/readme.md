# ArcSlider 示例

## 应用说明

演示 ArcSlider 控件（弧形滑块），4个不同尺寸弧形滑块在2x2网格中，拖动改变值。

## 控件列表

| 控件 | 类型 | 尺寸 (宽x高) | 初始值 |
|------|------|--------------|--------|
| arc_1 | egui_view_arc_slider_t | 90x55 | 20 |
| arc_2 | egui_view_arc_slider_t | 120x75 | 40 |
| arc_3 | egui_view_arc_slider_t | 55x55 | 60 |
| arc_4 | egui_view_arc_slider_t | 90x90 | 80 |
| grid | egui_view_gridlayout_t | 240x280 | - |

## 录制动作

| 序号 | 动作类型 | 目标控件 | 说明 |
|------|----------|----------|------|
| 0 | 拖动 | arc_1 | 从左下(0.2,0.8)拖到右上(0.8,0.2)，15步 |
| 1 | 拖动 | arc_2 | 从左下(0.2,0.8)拖到右上(0.8,0.2)，15步 |
| 2 | 拖动 | arc_3 | 从左下(0.2,0.8)拖到右上(0.8,0.2)，15步 |
| 3 | 拖动 | arc_4 | 从左下(0.2,0.8)拖到右上(0.8,0.2)，15步 |
