# ButtonImg 示例

## 应用说明

演示图片背景按钮，1个按钮(80x80)使用图片作为normal/pressed背景，点击时切换图片。

## 控件列表

| 控件 | 类型 | 尺寸 (宽x高) | 说明 |
|------|------|--------------|------|
| button_1 | egui_view_button_t | 80x80 | 图片背景按钮，normal使用test图片，pressed使用star图片 |

## 录制动作

| 序号 | 动作类型 | 目标控件 | 说明 |
|------|----------|----------|------|
| 0 | 点击 | button_1 | 点击按钮（显示pressed图片） |
| 1 | 点击 | button_1 | 再次点击按钮 |
