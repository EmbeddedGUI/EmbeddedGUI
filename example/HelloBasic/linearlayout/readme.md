# LinearLayout 控件演示

## 应用说明

演示 LinearLayout 线性布局控件的基本用法。使用一个全屏大小的线性布局容器，内部包含两个等高的 Label 子控件，分别设置绿色和橙色背景。布局方向为垂直排列，子控件居中对齐。

## 控件列表

| 变量名 | 控件类型 | 尺寸 (宽x高) | 说明 |
|--------|---------|-------------|------|
| layout_1 | LinearLayout | SCEEN_WIDTHxSCEEN_HEIGHT | 全屏线性布局容器，居中对齐 |
| label_1 | Label | SCEEN_WIDTHx(SCEEN_HEIGHT/2) | 文本 "Item1"，绿色背景 |
| label_2 | Label | SCEEN_WIDTHx(SCEEN_HEIGHT/2) | 文本 "Item2"，橙色背景 |

## 录制动作

| 序号 | 动作类型 | 间隔 | 说明 |
|------|---------|------|------|
| 1 | WAIT | 1500ms | 等待展示线性布局显示效果 |
