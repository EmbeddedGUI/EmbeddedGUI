# Button 控件演示

## 应用说明

演示 Button 控件基本用法，展示不同尺寸和样式的按钮。布局采用垂直居中排列。

## 控件列表

| 变量名 | 控件类型 | 尺寸 (宽x高) | 说明 |
|--------|---------|-------------|------|
| button_1 | Button | 140x30 | 宽型小按钮 |
| button_2 | Button | 200x45 | 宽型大按钮 |
| button_3 | Button | 80x50 | 紧凑型小按钮 |
| button_4 | Button | 120x70 | 紧凑型大按钮 |

## 录制动作

| 序号 | 动作类型 | 目标 | 间隔 | 预期效果 |
|------|---------|------|------|---------|
| 1 | CLICK | button_1 | 1000ms | 按钮显示按下状态 |
| 2 | CLICK | button_2 | 1000ms | 按钮显示按下状态 |
| 3 | CLICK | button_3 | 1000ms | 按钮显示按下状态 |
| 4 | CLICK | button_4 | 1000ms | 按钮显示按下状态 |
