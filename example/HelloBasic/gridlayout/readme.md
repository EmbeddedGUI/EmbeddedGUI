# GridLayout 示例

## 应用说明

演示 GridLayout 控件（网格布局），2x2网格中放置4个Switch开关，点击切换各开关状态。

## 控件列表

| 控件 | 类型 | 尺寸 (宽x高) | 初始状态 |
|------|------|--------------|----------|
| grid | egui_view_gridlayout_t | 200x200 | 2列网格布局 |
| sw_1 | egui_view_switch_t | 60x30 | 开启 |
| sw_2 | egui_view_switch_t | 60x30 | 关闭 |
| sw_3 | egui_view_switch_t | 60x30 | 关闭 |
| sw_4 | egui_view_switch_t | 60x30 | 开启 |

## 录制动作

| 序号 | 动作类型 | 目标控件 | 说明 |
|------|----------|----------|------|
| 0 | 点击 | sw_1 | 切换开关1状态 |
| 1 | 点击 | sw_2 | 切换开关2状态 |
| 2 | 点击 | sw_3 | 切换开关3状态 |
| 3 | 点击 | sw_4 | 切换开关4状态 |
