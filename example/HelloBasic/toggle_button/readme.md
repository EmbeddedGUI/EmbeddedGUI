# ToggleButton 示例

## 应用说明

演示 ToggleButton 控件（文字切换按钮），4个不同尺寸，带文字标签 Opt A/B/C/D，点击切换选中/未选中状态。

## 控件列表

| 控件 | 类型 | 尺寸 (宽x高) | 文字 | 初始状态 |
|------|------|--------------|------|----------|
| toggle_1 | egui_view_toggle_button_t | 90x25 | Opt A | 未选中 |
| toggle_2 | egui_view_toggle_button_t | 140x35 | Opt B | 选中 |
| toggle_3 | egui_view_toggle_button_t | 60x35 | Opt C | 未选中 |
| toggle_4 | egui_view_toggle_button_t | 90x50 | Opt D | 选中 |

## 录制动作

| 序号 | 动作类型 | 目标控件 | 说明 |
|------|----------|----------|------|
| 0 | 点击 | toggle_1 | 切换toggle_1状态 |
| 1 | 点击 | toggle_2 | 切换toggle_2状态 |
| 2 | 点击 | toggle_3 | 切换toggle_3状态 |
| 3 | 点击 | toggle_4 | 切换toggle_4状态 |
